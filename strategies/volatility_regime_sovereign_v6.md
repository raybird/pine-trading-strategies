# TN Volatility Regime Sovereign v6 (波動率環境主權版)

## 策略概述
這是一個具備「環境自覺」的主權策略。它不直接預測價格，而是先透過歸一化 ATR 偵測市場的波動率環境。根據環境的「有序」與「混沌」程度，系統會自動切換不同的執行規訓（如調整倉位或暫停交易）。這是 TeleNexus 用於應對「黑天鵝」或「流動性枯竭」的核心保護機制。

## 核心特性
- **波動率環境感知 (Regime Detection)**: 透過 `(ATR / Price) * 100` 的歸一化演算，將市場區分為三種主權狀態：
  1. **Quiet (平穩期)**: 低波動均值回歸環境。
  2. **Active (活躍期)**: 高波動趨勢追隨環境。
  3. **Extreme (極端期)**: 系統判定為不可控風險，觸發「主權避險」，禁止一切交易。
- **動態倉位規訓 (Dynamic Sizing)**: 利用 v6 `switch` 語句直接映射環境狀態與倉位比例（15% 到 0%），實現風險與環境的自動對位。
- **主權避險機制 (Risk-Off)**: 當波動率指標突破 `highThresh` 時，系統自動清空部位並進入觀察模式，保護執行主權不被市場無序性侵蝕。
- **環境熱圖看板**: 透過 Table API 實時披露當前的歸一化波動率百分比與活躍的規訓模式。

## 模組說明
1. **Regime Classifier**: 核心感知模組，負責環境定性。
2. **Position Controller**: 動態計算符合當前環境的主權倉位。
3. **Execution Logic**: 結合 EMA 交叉作為基準載體。
4. **Environment HUD**: 提供視覺化的環境狀態反饋。

## 使用建議
- **主週期**: 適用於 1H 到 Daily 週期。
- **規訓用途**: 適合作為全域組合的「波動率過濾器」，防止在極端不穩定時期過度曝險。
