# TN Sovereign Fakeout Filter v6

## 1. 策略概述
**TN Sovereign Fakeout Filter v6** 是一款基於「四層因果審計」的高階量化執行策略。其核心目標在於透過多維度的環境感知，過濾市場中的隨機噪聲與假突破（Fakeouts），確保每一筆交易行為都具備高度的執政主權（Sovereignty）與執行一致性。

- **版本**: v6.0 (Pine Script v6)
- **實體化日期**: 2026-04-15 (台灣時間 CST)
- **核心架構**: 4-Layer Causal Audit (Volume + Bar Action + HTF Anchor + ATR Noise)

## 2. 四層因果審計邏輯

### 第一層：成交量審計 (F1: Volume Confirmation)
- **機制**: 當前成交量必須大於 20 日移動平均成交量的 1.5 倍。
- **目的**: 驗證突破行為是否由機構資金（物理足跡）驅動，排除低流動性引發的隨機波動。

### 第二層：價格行為硬化 (F2: Bar Confirmation)
- **機制**: 均線交叉信號發出後，需等待 N 根 K 線確認價格維持在交叉方向。
- **目的**: 防止「秒級穿刺」導致的頻繁止損，確保趨勢動能已在當前時框硬化。

### 第三層：宏觀因果錨點 (F3: HTF Trend Alignment)
- **機制**: 強制要求執行方向必須與 60 分鐘（1H）時框的 50 日 EMA 一致。
- **目的**: 確保微觀執行與宏觀慣性共振，避免在震盪市中進行逆勢博弈。

### 第四層：波動率噪聲壓制 (F4: ATR Noise Filter)
- **機制**: 當前 K 線的實體變動必須大於 ATR 的 0.3 倍。
- **目的**: 過濾「縮量盤整」期間的微小位移，僅捕捉具備結構性因果強度的信號。

## 3. 風險規訓 (Risk Management)
- **動態倉位分配**: 根據 ATR (14) 自動演算止損距離（2.0x ATR），並依據固定 1.0% 的風險權重分配下單單位。
- **執行透明度**: 在圖表右上角實時顯示「因果審計狀態表」，確保 Agent 的執行動機隨時可受物理監督。
