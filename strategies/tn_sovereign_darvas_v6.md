# TN Sovereign Darvas Box v6

## 1. 策略概述
**TN Sovereign Darvas Box v6** 是一款基於經典尼古拉斯·達瓦斯（Nicolas Darvas）箱體理論的高階自動化策略。其核心目標在於識別市場的物理邊界，並透過「重新進場規訓（Re-entry Protocol）」——即價格在掃蕩邊界後重新回到箱體內——來捕捉高品質的趨勢延續機會。

- **版本**: v6.0 (Pine Script v6)
- **實體化日期**: 2026-04-15 (台灣時間 CST)
- **核心架構**: Automated Box Detection + Re-entry Protocol + HTF Causal Anchor

## 2. 技術邏輯分析

### 第一層：自動化箱體感知 (Automated Box Detection)
- **機制**: 透過設定樞軸長度（Box Pivot Length）偵測局部高低點，動態更新箱體的上軌（Upper）與下軌（Lower）。
- **目的**: 定義市場的短期平衡區間，作為後續因果判定基準。

### 第二層：重新進場規訓 (Re-entry Protocol)
- **武裝（Armed）**: 當價格收盤於箱體外部時，系統進入「預備狀態」。
- **觸發（Signal）**: 當價格在武裝狀態後，重新收盤於箱體內部，視為「虛假突破後的動能回歸」，觸發執行。
- **目的**: 利用市場的掃蕩行為獲取更好的進場價格，排除傳統突破策略容易遭遇的假突破風險。

### 第三層：宏觀因果錨點 (HTF Alignment)
- **機制**: 強制要求執行方向必須與 4 小時時框（240m）的 200 日 EMA 一致。
- **規訓**: 僅在宏觀趨勢共振時執行，建立執行的物理地板。

## 3. 風險規訓 (Risk Management)
- **動態止損**: 以 ATR (14) 或箱體邊界為基準計算止損，單筆風險嚴格鎖定在 1.0%。
- **盈虧對位**: 設定 1:3 的 Reward/Risk 比率，確保長期期望值的穩定性。

## 4. 執行說明
本策略已全面對焦 Pine Script v6 標準，並執行嚴格的物理時間對位（v26.0415.*）。建議執行於 1 小時（1H）或 15 分鐘（15m）時框，並實時監控 Dashboard 中的「武裝狀態」。
