# Tn Sovereign Liquidity Matrix v6

## 策略概述
這是一款基於 **Adaptive Liquidity Matrix (ALM)** 框架深度重構的主權策略，採用 Pine Script v6.0 標準構建。其核心理念在於「市場結構的動態規訓」，透過 MTF 極性引擎與自適應 ATR 斐波那契矩陣，自動識別市場的供需地板，並在流動性交互點執行高勝率的因果交易。

## 核心特性
- **矩陣核心引擎 (Matrix Core Engine)**:
    - **宏觀定標**: 自動掃描指定時框（如 Daily）的最高成交量區域，確立市場的物理因果起點（Anchor）。
    - **自適應斐波那契擴展**: 基於宏觀 ATR 動態生成多層供需矩陣，精確定義市場的執行邊界。
- **因果訊號合成 (Causal Signal Synthesis)**:
    - **流動性交互審計**: 監測價格與矩陣邊界的物理接觸，僅在發生明顯拒絕（Rejection）時觸發進場指令。
    - **非重繪規訓**: 全量採用非重繪的 MTF 數據獲取邏輯，確保回測軌跡與實戰一致。
- **主權審計 HUD**: 在圖表右下角即時披露宏觀錨點、極性狀態、以及與下一供需目標的距離，達成執行過程的完全透明化。

## 模組說明
1. **Matrix Engine**: 負責市場結構的定標與矩陣生成。
2. **Signal Discipline**: 管理流動性交互邏輯與執行參數。
3. **Sovereign HUD**: 提供實時執行狀態審計與版本定標。

## 使用建議
- **推薦標的**: 股指、外匯、或具備高度流動性的加密貨幣（BTC, ETH）。
- **推薦時框**: 5M 或 15M 為執行時框，搭配 Daily 背景。
- **版本號**: v26.0420.1802
