# TN Sovereign MTF Dual-Layer v6

## 1. 策略概述
**TN Sovereign MTF Dual-Layer v6** 是一款基於「雙層嵌套因果審計」的高階量化執行策略。其核心目標在於透過宏觀趨勢的「軟鎖定」來引導微觀時框的「硬執行」，確保每一筆交易都具備跨時框的物理共振。

- **版本**: v6.0 (Pine Script v6)
- **實體化日期**: 2026-04-15 (台灣時間 CST)
- **核心架構**: Macro Trend Validation (5m) + Micro Signal Execution (1m)

## 2. 雙層審計邏輯

### 第一層：宏觀趨勢驗證 (Layer 1: Macro Trend)
- **週期**: 預設為 5 分鐘（5m）。
- **機制**: 同時校準 9/21 EMA 交叉方向與 Supertrend (3, 10) 信號。
- **目的**: 建立「因果地板」。只有當宏觀趨勢處於絕對多頭（均線多頭排列且 Supertrend 為買入）時，才允許微觀做多。

### 第二層：微觀進場執行 (Layer 2: Micro Entry)
- **週期**: 預設為圖表時框（建議 1m）。
- **信號**: 採用 EMA 9/21 交叉作為觸發器。
- **物理過濾**: 整合成交量審計（Vol > SMA 20 * 1.3）與 ATR 噪聲過濾（變動需達 0.3x ATR）。

## 3. 風險規訓 (Risk Management)
- **主權權益分配**: 根據帳戶權益自動演算下單單位，將單筆風險嚴格鎖定在 1.0%。
- **動態止損**: 採用 2.0x ATR 的止損距離與 4.0x ATR 的止盈目標，實現高度不對稱的期望值。

## 4. 執行說明
本策略已全面對焦 Pine Script v6 標準，並執行嚴格的物理時間對位（v26.0415.*）。建議執行於 1 分鐘（1m）圖表時框，並開啟「Dashboard」以實時監控宏觀環境與微觀信號的對位狀態。
