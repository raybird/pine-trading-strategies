# TN Sovereign v6 (核心主權版)

## 策略概述
這是 TeleNexus 主權交易系列的基礎框架策略。它旨在演示如何在 Pine Script v6 中實作「多層因果審計」，並結合強型別管理（Enum）與非重繪的高時框過濾。

## 核心特性
- **強型別規訓**: 使用 `enum TrendState` 與 `SignalGrade` 管理交易狀態，確保邏輯的語義嚴謹性。
- **4 層因果過濾矩陣 (Audit Score)**: 
  1. **成交量對位**: 確保當前動能具備物理支撐。
  2. **HTF 趨勢審計**: 透過 D1 或 H4 週期確認宏觀因果方向。
  3. **ATR 噪聲過濾**: 排除低波動盤整期的隨機性。
  4. **EMA 結構排列**: 確保局部趨勢具備連續性。
- **物理時間對位**: 透過 `[1]` 偏移量獲取高時框數據，確保歷史回測與實盤執行的因果一致性（杜絕重繪）。

## 模組說明
1. **數據獲取層**: 執行 v6 動態 `request.security` 審計。
2. **評分引擎**: 將 4 層過濾結果轉化為 0-4 分的主權分值。
3. **執行層**: 僅在主權分值達標 (Sovereign/Strong) 時啟動執行。

## 使用建議
- **主週期**: 15M 或 1H。
- **過濾週期**: Daily。
- **版本號**: v26.0408.1830
