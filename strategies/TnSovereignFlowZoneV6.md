# Tn Sovereign FlowZone v6

## 策略概述
這是一款基於 **FlowZone** 框架深度重構的主權策略，採用 Pine Script v6.0 標準構建。其核心理念在於「多維因果匯流」，透過對趨勢、動能與物理能量（成交量）的加權審計，計算出即時的「匯流分數 (Confluence Score)」。只有當市場多項指標產生共振且突破定標門檻時，系統才會發出執行指令。

## 核心特性
- **因果匯流引擎 (Confluence Engine)**:
    - **趨勢審計**: 監測 200 EMA 宏觀地板，確立多空基調。
    - **動能審計**: 利用 RSI 與均線交叉進行動能確認。
    - **能量審計**: 實施成交量激增檢查，確保匯流具備物理支撐。
- **動態匯流評分**: 系統即時演算匯流分數（0-10分），並根據「匯流門檻」動態過濾無效噪音。
- **結構化風險規訓**:
    - 採用 ATR 動態止損，確保風險曝險與市場波動率精確對位。
    - 實作多級盈虧比目標，達成科學的資本保護。
- **實時審計 HUD**: 在圖表右上角披露匯流分數、宏觀環境狀態與能量審計結果，達成執行過程的 100% 透明化。

## 模組說明
1. **Confluence Core**: 負責加權分數演算與匯流門檻規訓。
2. **Risk Management**: 管理 ATR 止損與多級執行參數。
3. **Causal Execution**: 執行信號合成與訂單生命週期管理。
4. **Audit HUD**: 披露實時執行數據與版本定標。

## 使用建議
- **推薦標的**: 指數、商品期貨、高波動率加密貨幣。
- **推薦時框**: 5M 或 15M (執行) 搭配 1H (背景)。
- **版本號**: v26.0421.1206
