# TN ML Matrix Ensemble Sovereign v6 (機器學習矩陣集成版)

## 策略概述
此策略代表了 TeleNexus 對「非線性市場預測」的深度探索。它結合了 Lorentzian 空間距離演算（用於捕捉非線性模式）與傳統的策略過濾矩陣（趨勢、動能、波動率），透過「集成共識 (Ensemble Consensus)」機制，實現了具備高魯棒性的主權級決策。

## 核心特性
- **Lorentzian 距離預測引擎**: 不同於傳統線性回歸，此引擎計算多維特徵（RSI, CCI, ADX, Mom）在非歐幾里得空間（Lorentzian Space）的離散度，用以捕捉市場的極端情緒與反轉節點。
- **策略過濾矩陣 (Strategy Matrix)**: 整合了宏觀趨勢、相對動能與波動率密度的物理審計，確保機器學習的預測結果具備傳統技術分析的底層支撐。
- **三級集成共識 (Consensus Modes)**:
  1. **Strict (0.8)**: 要求 ML 預測與策略矩陣達成高度同步，僅在極強共識下進場。
  2. **Balanced (0.6)**: 標準規訓模式，適合多數流動性市場。
  3. **Aggressive (0.4)**: 捕捉微弱的早期同步信號。
- **自適應物理退出**: 基於 ATR 的動態風險管理，預設提供 2:1 的初始盈虧比規訓。

## 模組說明
1. **Feature Engineering**: 實作自定義 `normalize` 方法，將不同維度的技術指標對位至統一的感知區間。
2. **Lorentzian Engine**: 執行非線性空間演算並輸出 `ml_score`。
3. **Strategy Matrix Auditor**: 負責傳統技術指標的綜合評分。
4. **Ensemble Dashboard**: 透過桌表 API 實時對比 ML 預測值與矩陣評分，展示系統的共識狀態。

## 使用建議
- **主週期**: 建議在 30M 或 1H 週期使用。
- **適用環境**: 特別適合趨勢轉折點的偵測與波動率擴張初期的介入。
