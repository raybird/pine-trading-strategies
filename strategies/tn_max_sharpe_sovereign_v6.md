# TN Max Sharpe Sovereign v6 (最大夏普比率主權優化版)

## 策略概述
此策略致力於解決多資產配置中的「效能最優化」問題。它在 Pine Script v6 中實作了馬科維茨切線投資組合 (Tangent Portfolio) 的矩陣求解。透過動態演算資產間的超額收益向量與協方差矩陣，系統自動計算出能提供最大夏普比率（風險調整後收益最大化）的資產權重，實現具備數學主權的資金分配。

## 核心特性
- **矩陣化 Sharpe 求解器**: 利用 v6 `matrix.inverse()` 與 `matrix.mult()` 方法，動態求解方程式 $w = \Sigma^{-1} \mu$，精確鎖定最優風險收益比的坐標。
- **動態優化窗口 (Optimization Windows)**:
  1. **ShortTerm (60 Bars)**: 快速捕捉短期熱點切換。
  2. **Standard (120 Bars)**: 兼顧統計穩定性與時效性。
  3. **LongTerm (252 Bars)**: 宏觀主權配置，鎖定年度級別的優勢資產。
- **物理對位收益演算**: 使用自定義 `method` 獲取實體資產的對數收益率，並扣除動態設定的無風險利率 (Risk-Free Rate)，確保因果審計的真實性。
- **自動化主權對齊**: 策略會根據當前圖表所屬資產，自動從最優權重矩陣中提取對應的 `qty` 進行執行。

## 模組說明
1. **Asset Sensor**: 負責多資產同步數據感知。
2. **Matrix Engine**: 執行高階代數運算，負責求解 $\Sigma$ 與 $\mu$。
3. **Weight Normalizer**: 將矩陣解轉化為符合主權風險限制的實體權重。
4. **Diagnostic Dashboard**: 實時披露各資產的優化權重分佈。

## 使用建議
- **主週期**: 建議在 Daily 或 4H 週期使用，以累積具備回歸意義的收益數據。
