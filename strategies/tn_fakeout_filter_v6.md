# TN Sovereign Fakeout Filter v6

## 1. 策略概述
**TN Sovereign Fakeout Filter v6** 是一款基於「四層因果審計」的量化執行策略。其核心目標在於透過多維度的環境感知，過濾市場中的隨機噪聲與假突破（Fakeouts），確保交易行為具備高度的執法主權（Sovereignty）。

- **版本**: v6.0 (Pine Script v6)
- **實體化日期**: 2026-04-14 (台灣時間 CST)
- **核心架構**: 4-Layer Causal Scoring (Volume + Trend + Noise + Alignment)

## 2. 四層因果過濾邏輯

### 第一層：成交量審計 (Volume Confirmation)
- **機制**: 當前成交量必須大於 20 日均量的 1.5 倍。
- **目的**: 確保突破具備真實的物理動能，排除低流動性的誘多/誘空行為。

### 第二層：高時框趨勢對位 (HTF Trend Alignment)
- **機制**: 價格必須位於 60 分鐘 K 線（1H）的 50 日 EMA 之上。
- **目的**: 確保微觀執行與宏觀慣性一致，降低逆勢交易風險。

### 第三層：ATR 噪聲壓制 (ATR Noise Filter)
- **機制**: 單根 K 線的變動幅度必須大於 ATR 的 0.3 倍。
- **目的**: 過濾震盪行情中的微小波動，捕捉具備結構性位移的信號。

### 第四層：均線物理對齊 (EMA Alignment)
- **機制**: 快線 EMA (9) 必須位於慢線 EMA (21) 之上方。
- **目的**: 確認短期趨勢的動能方向已完全硬化。

## 3. 風險規訓 (Risk Management)
- **動態倉位控制**: 根據 ATR (14) 自動演算止損距離（1.5x ATR），並依據 1.0% 固定風險分配下單單位。
- **執行透明度**: 在圖表右上角實時顯示「主權得分 (Sovereign Score)」，得分 >= 3 方可視為穩定（STABLE）信號。

## 4. 執行規訓
建議於 15 分鐘或 30 分鐘時框執行，以平衡信號靈敏度與趨勢過濾的準確性。本策略已全面對焦 Pine Script v6 標準，具備高效的數據讀取與非重繪（Non-repainting）特性。
