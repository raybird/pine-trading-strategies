# Tn Sovereign Triple EMA v6

## 策略概述
這是一款基於經典 **Triple EMA (5/13/34)** 多重對位邏輯深度重構的主權策略，採用 Pine Script v6.0 標準構建。其核心理念在於「分層因果對位」：將快速均線的排列視為局部趨勢信號，並透過 200 EMA 全球地板進行宏觀環境過濾。這確保了系統僅在短中長三週期趨勢產生物理共振時才執行交易，極大地提升了趨勢跟蹤的穩定性。

## 核心特性
- **三均線同步規訓 (Alignment Core)**:
    - 實作 5/13/34 EMA 的順序排列審計（Bull/Bear Alignment）。
    - 只有在對位關係首次確立（ ta.change 偵測）時觸發進場，避免在震盪區間反覆切換。
- **全球環境地板 (Global Regime Audit)**:
    - 引入 200 EMA 作為多空執行門檻，自動屏蔽逆勢操作。
    - 確保所有局部信號都具備深厚的全球因果支持。
- **物理能量審計 (Energy Audit)**:
    - 整合成交量審計（Volume vs SMA 20），確保均線對位伴隨著真實的市場參與度。
- **實時審計 HUD**: 在圖表右上角即時披露均線同步狀態、全球環境偏見、能量審計結果以及版本定標。

## 模組說明
1. **EMA Config**: 負責快、中、慢三條均線的參數規訓。
2. **Audit Layers**: 執行 200 EMA 環境過濾與成交量能量審計。
3. **Causal Signals**: 整合訊號並管理訂單生命週期（對位消失即出場）。
4. **Audit HUD**: 披露執行狀態與版本定標。

## 使用建議
- **推薦標的**: 指數、趨勢穩定的權值股、主流加密貨幣。
- **推薦時框**: 15M 或 30M (執行) 搭配 4H (環境)。
- **版本號**: v26.0422.0011
