# TN Multilateral Sovereign Corr v6 (多邊主權相關性矩陣版)

## 策略概述
這是一個專業級的市場感知工具，專注於監測「因果脫鉤 (Decoupling)」。它在 Pine Script v6 中利用矩陣運算與循環請求技術，同步計算一個資產集群（如加密貨幣組或宏觀組）內部的皮爾森相關係數矩陣。當預設的資產對出現異常的相關性失效時，觸發主權對沖信號。

## 核心特性
- **動態資產集群 (Asset Clusters)**: 支援預設的 Crypto (BTC, ETH, SOL, BNB) 與 Macro (GOLD, OIL, DXY, SPX) 矩陣，或由使用者手動自定義符號。
- **矩陣熱圖視覺化 (Correlation Heatmap)**: 利用 v6 Table API 實體化一個動態矩陣圖，以紅綠色塊直觀展示資產間的耦合強度。
- **因果脫鉤偵測 (Decoupling Engine)**: 捕捉相關性跌破關鍵閾值的瞬間。這通常預示著特定資產的基本面發生了獨立於大盤的結構性變化。
- **原生循環審計**: 展示了如何在 `barstate.islast` 階段利用 v6 效能優勢執行大規模的多標的同步數據獲取。

## 使用建議
- **主週期**: 建議在 1H 或更長週期使用。
- **規訓用途**: 適合作為大型量化工作室的「全域監控牆」，用於輔助判斷何時應切換對沖策略。
