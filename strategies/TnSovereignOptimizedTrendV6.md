# 🚀 TN Sovereign Optimized Trend v6

## 🚀 策略概述
`TN Sovereign Optimized Trend v6` 是一款基於 **Pine Script v6** 標準構建的進階趨勢跟隨策略。它結合了多尺度移動平均線 (EMA)、RSI 強度確認以及成交量噴發過濾器，旨在環境劇烈波動時精確捕捉具備機構支持的趨勢方向。

## 🧠 核心邏輯 (Causal Logic)
1.  **多尺度趨勢過濾**：使用 200 EMA 作為「宏觀因果錨點」，確保所有交易均符合大週期趨勢背景。
2.  **動態成交量審計**：進場信號必須伴隨成交量超過 20 周期均值的特定倍數（預設 1.2 倍），以過濾缺乏流動性支持的虛假交叉。
3.  **動態風險對位**：採用基於 ATR 的止損與倉位計算，自動適應市場的波動率範式 (Volatility Regimes)。
4.  **v6 規訓實裝**：全面遵循 v6 語法，移除 `when` 參數，改用明確的布林邏輯與 `if` 條件塊，提升執行效率與可觀測性。

## ⚙️ 輸入參數
- **EMA Fast/Slow**: 趨勢交叉的快慢線週期。
- **Macro Filter EMA**: 長線趨勢過濾邊界。
- **Volume Surge Mult**: 機構進場確認的成交量倍數。
- **Risk % per Trade**: 基於主權資產比例的風險控制。
