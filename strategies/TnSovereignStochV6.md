# Tn Sovereign Stoch v6

## 策略概述
這是一款基於 **Stochastic (隨機指標)** 深度重構的主權策略，採用 Pine Script v6.0 標準構建。其核心理念在於「動量共振與趨勢對位」：系統在隨機指標進入極端超買或超賣區域後尋找交叉，並輔以 200 EMA 趨勢地板進行過濾。這確保了系統僅在市場展現出強大反轉動能且具備宏觀因果支持時，才會執行交易指令。

## 核心特性
- **v6 原生動能對位**:
    - 採用 Pine Script v6 的 `ta.stoch` 邏輯，結合自定義的 K/D 平滑演算法。
    - 實施區域審計：僅在超買（>80）或超賣（<20）區域發生的交叉被視為高強度信號。
- **宏觀環境規訓**:
    - 整合 200 EMA 作為「因果界線」，自動屏蔽逆向操作（如在空頭背景下做多）。
    - 確保所有反轉交易都具備長期趨勢的支撐。
- **動態風險對位**:
    - 採用基於 ATR 的波動率對位算法，自動優化止損與獲利目標。
- **實時審計 HUD**: 在圖表右上角即時披露 K/D 數值、市場偏見（BULL/BEAR）以及實時波動率，達成執行過程的 100% 透明。

## 模組說明
1. **Stochastic Discipline**: 負責隨機指標的計算與區域規訓。
2. **Audit Layers**: 執行趨勢地板過濾與 ATR 風險對位。
3. **Causal Signals**: 整合訊號並管理訂單生命週期。
4. **Audit HUD**: 披露實時執行狀態與版本定標。

## 使用建議
- **推薦標的**: 外匯、權值股、具備週期性特徵的商品。
- **推薦時框**: 15M 或 1H (執行) 搭配 4H (環境)。
- **版本號**: v26.0422.1203
