# Tn Sovereign VWAP v6

## 策略概述
這是一款基於 **VWAP (成交量加權平均價)** 深度重構的主權均值回歸策略，採用 Pine Script v6.0 標準構建。系統核心在於捕捉價格相對於「機構平均持倉成本」的極端偏離：透過計算動態標準差通道，並結合 RSI 超買超賣審計與 200 EMA 趨勢地板，確保交易指令僅在價格展現出明確的「超跌反彈」或「超漲修正」意圖時觸發。

## 核心特性
- **動態錨定 VWAP (Anchored VWAP)**:
    - 支援 Session、Day、Week 三種錨定週期，自動對位不同時間維度的機構均衡位。
    - 實施物理標準差通道 (1.0 - 3.0x)，精確量化價格的「因果偏離度」。
- **雙重能量審計 (Momentum Audit)**:
    - 整合 RSI 指標，要求價格在穿透 VWAP 邊界時必須伴隨動能衰竭信號，過濾掉強勁趨勢中的「順向突破」。
- **宏觀環境規訓 (Regime Audit)**:
    - 強制整合 200 EMA 作為趨勢地板，確保所有逆向交易均具備深厚的宏觀物理合理性。
- **實時審計 HUD**: 在圖表右上角即時披露價格相對於 VWAP 的偏離百分比、RSI 狀態、市場偏見以及版本定標，達成執行主權的完全透明化。

## 模組說明
1. **VWAP Config**: 負責錨定週期與標準差倍數的規訓。
2. **Audit Layers**: 執行趨勢地板過濾、RSI 動能審計與風險對位。
3. **Causal Signals**: 整合訊號並管理訂單生命週期（以 VWAP 中軌作為目標獲利位）。
4. **Audit HUD**: 披露實時執行數據與版本定標。

## 使用建議
- **推薦標的**: 高流動性藍籌股、股指期貨、主流加密貨幣。
- **推薦時框**: 5M 或 15M (執行) 搭配 1H (環境)。
- **版本號**: v26.0426.1831
