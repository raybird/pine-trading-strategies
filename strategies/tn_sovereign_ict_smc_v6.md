# TN Sovereign ICT SMC v6

## 1. 策略概述
**TN Sovereign ICT SMC v6** 是一款融合了「聰明錢概念 (Smart Money Concepts)」與「因果時間對位」的高階量化策略。其核心在於識別市場結構位移 (BOS)、流動性掃蕩與訂單流失衡 (FVG)，並僅在機構級交易時段（倫敦與紐約殺戮區）執行，確保每一筆交易都具備最高的主權頻率與因果確定性。

- **版本**: v6.0 (Pine Script v6)
- **實體化日期**: 2026-04-16 (台灣時間 CST)
- **核心架構**: BOS/CHOC + FVG Imbalance + Institutional Killzones

## 2. 因果審計邏輯

### 第一層：市場結構與 BOS (Market Structure)
- **機制**: 利用樞軸點 (Pivot High/Low) 識別趨勢的結構性高低點。
- **判定**: 當價格有效突破前一個結構極值時，觸發 BOS (Break of Structure)，定義系統的因果方向偏好。

### 第二層：流動性失衡與 FVG (Fair Value Gap)
- **機制**: 偵測 K 線間的物理空隙（失衡區）。
- **目的**: 識別機構資金留下的足跡。失衡區被視為價格必然回踩的「引力點」。

### 第三層：訂單塊審計 (Order Blocks)
- **判定**: 識別在趨勢反轉前最後一根具備「流動性掃蕩」特徵的 K 線。
- **對位**: 只有當價格在 FVG 回踩與 OB 區間達成物理對位時，才具備進場主權。

### 第四層：因果時間對位 (Killzones)
- **倫敦時段**: CST 15:00 - 18:00。
- **紐約時段**: CST 21:00 - 00:00。
- **規訓**: 嚴禁在非機構時段執行，以規避散戶盤整導致的隨機噪聲。

## 3. 風險規訓 (Risk Management)
- **動態倉位分配**: 根據 ATR (14) 計算止損距離，確保單筆交易風險嚴格控制在 1.0%。
- **期望值優化**: 設定 1:2.5 的 Reward/Risk 比率，實現高度不對稱的回報矩陣。

## 4. 執行說明
本策略已全面對焦 Pine Script v6 標準，並執行嚴格的物理時間對位（v26.0416.*）。建議執行於 5 分鐘 (5m) 或 15 分鐘 (15m) 時框，以獲得最佳的結構捕捉精度。
