# Tn Sovereign Supertrend v6

## 策略概述
這是一款基於經典 **Supertrend (超級趨勢)** 深度重構的主權策略，採用 Pine Script v6.0 標準與原生 `ta.supertrend` 函數構建。其核心在於「多維趨勢對位」，透過引入 200 EMA 與 50 EMA 作為物理趨勢地板，確保系統僅在市場展現出強大且一致的因果動能時才執行進場指令，大幅提升了趨勢交易的勝率與穩定性。

## 核心特性
- **多維趨勢審計 (Multi-Regime Audit)**:
    - **200 EMA 地板**: 作為長線多空的物理分界，自動過濾掉與大趨勢相悖的噪音。
    - **50 EMA 導向**: 作為中線動能的輔助確認，確保進場時市場具備足夠的加速慣性。
- **原生 v6 動力學**:
    - 全量採用 Pine Script v6.0 的 `ta.supertrend` 實作，具備極高的運算效率與非重繪的精確度。
    - 利用 v6 的 `input` 分組功能，實現工業級的參數規訓管理。
- **實時審計 HUD**: 在圖表右上角即時披露超級趨勢方向、多維環境偏見（BULL-BIAS/BEAR-BIAS/NEUTRAL）以及實時波動率（ATR），達成執行主權的完全透明化。

## 模組說明
1. **Supertrend Core**: 負責波動率止損線的生成與趨勢轉向偵測。
2. **Regime Audit**: 管理 200/50 EMA 的環境過濾邏輯。
3. **Causal Signals**: 整合環境與動能，產出精確的執行指令。
4. **Audit HUD**: 披露實時執行狀態與版本定標。

## 使用建議
- **推薦標的**: 股指、大盤成長股、主流加密貨幣。
- **推薦時框**: 30M 或 1H (執行) 搭配 4H (環境)。
- **版本號**: v26.0421.1803
