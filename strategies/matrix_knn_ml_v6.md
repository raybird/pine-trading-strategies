# TN Matrix kNN ML Sovereign v6 (矩陣 KNN 機器學習版)

## 策略概述
此策略將機器學習中的「k-最近鄰居 (k-Nearest Neighbors)」演算法直接實體化於 Pine Script 運行時。它不依賴外部模型，而是透過 v6 的 `matrix` 物件在本地維護一個多維特徵空間（包含強弱指標、波動率與動能），實時計算當前市場狀態與歷史樣本的歐幾里得距離，從而預測未來的價格極性。

## 核心特性
- **本地特徵空間矩陣 (Matrix Feature Space)**: 全量利用 `matrix<float>` 儲存數百根 K 線的歸一化特徵與對應結果，實現「執行即訓練」的動態演化。
- **歐幾里得距離演算 (Euclidean Distance Engine)**: 透過自定義方法遍歷矩陣，在 3D 特徵空間中尋找最相似的歷史節點，計算市場的因果信心分值。
- **三級機器學習規訓 (ML Strictness)**:
  1. **Loose (0.4)**: 激進預測，適合趨勢早期的先行捕捉。
  2. **Balanced (0.6)**: 標準規訓，平衡勝率與盈虧比。
  3. **Strict (0.8)**: 主權級信心要求，僅在歷史模式高度對位時才觸發執行。
- **混合執行架構**: 將「ML 模式辨識」作為先驗條件，結合「EMA 均線交叉」作為物理觸發，確保交易決策兼具數據預測與趨勢確認。

## 模組說明
1. **Feature Engineering**: 執行特徵歸一化規訓（0-1 區間），消除不同量綱間的計算偏差。
2. **Matrix Manager**: 負責矩陣的動態滾動更新，確保特徵空間的時效性。
3. **kNN Predictor**: 核心機器學習演算模組，輸出 `ml_confidence`。
4. **Discipline HUD**: 即時披露 ML 信心指標與當前規訓強度。

## 使用建議
- **主週期**: 建議在 1H 或 4H 週期使用，以累積具備統計意義的特徵空間。
- **回顧長度**: `lookback` 建議設為 200 以上，以提升距離演算的精確度。
