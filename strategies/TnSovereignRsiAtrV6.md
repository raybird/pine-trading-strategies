# Tn Sovereign RSI ATR v6

## 策略概述
這是一款結合了 **Relative Strength Index (RSI)** 動能震盪與 **Average True Range (ATR)** 動態風險規訓的主權策略，採用 Pine Script v6.0 標準構建。系統核心在於「動能與環境的雙重審計」：只有當 RSI 展現出強大的相對強度且同時獲得 200 EMA 宏觀地板的物理確認時，系統才會執行進場指令，並利用 ATR 實施精確的風險對位。

## 核心特性
- **動能區域規訓 (Momentum Zone Discipline)**:
    - 實施嚴格的 RSI 門檻過濾，確保進場時市場具備足夠的趨勢加速性。
    - 做多：RSI > 60；做空：RSI < 40。
- **宏觀環境審計 (Regime Audit)**:
    - 整合 200 EMA 作為「因果分界線」，自動屏蔽逆向操作。
    - 確保所有動量交易均具備深厚的宏觀因果支持，有效過濾震盪噪音。
- **動態風險對位 (Dynamic Risk Alignment)**:
    - 採用基於 ATR 的波動率算法，自動計算止損 (2.0x ATR) 與獲利 (4.0x ATR) 空間。
    - 實現 1:2 的最小期望盈虧比規訓，達成科學的資本保護。
- **實時審計 HUD**: 在圖表右上角即時披露 RSI 數值、環境偏見（BULL/BEAR）、波動率水平以及版本定標，達成執行主權的完全透明化。

## 模組說明
1. **RSI Config**: 管理動能指標的週期與進場門檻。
2. **Audit Layers**: 執行趨勢地板過濾與 ATR 風險規訓。
3. **Causal Signals**: 整合訊號並管理訂單生命週期。
4. **Audit HUD**: 披露實時執行數據與版本定標。

## 使用建議
- **推薦標的**: 外匯、商品期貨、高流動性加密貨幣。
- **推薦時框**: 30M 或 1H (執行) 搭配 4H (環境)。
