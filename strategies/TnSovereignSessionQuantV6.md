# Tn Sovereign Session Quant v6

## 策略概述
這是一款專注於「時間與空間對位」的區間突破策略，採用 Pine Script v6.0 標準構建。其核心理念在於市場能量在特定地理時段（如台北或紐約開盤）的集中釋放。系統透過動態監測時段內的高低點位，並結合趨勢地板進行因果審計，確保系統只在發生結構性位移（Breakout）且具備方向一致性時才發出執行指令。

## 核心特性
- **地理時段規訓 (Session Discipline)**:
    - 支援動態配置執行時段與時區，實現對全球市場開盤能量的精準捕捉。
    - 實作「時段結束強制平倉」規訓，隔離隔夜風險，確保執行主權的日結。
- **區間突破審計 (Range Breakout Audit)**:
    - 自動追蹤時段內的高低點。
    - 只有在收盤價突破時段極值且獲得 EMA 趨勢確認時觸發。
- **多維環境規訓**:
    - 整合 50 EMA 作為基準均線，自動過濾掉無趨勢支撐的虛假突破。
- **實時審計 HUD**: 在圖表右上角即時披露當前時段狀態、區間極值、環境偏見以及版本定標，達成執行主權的完全透明化。

## 模組說明
1. **Session Config**: 負責執行時段、時區與平倉規則的規訓。
2. **Audit Layers**: 執行區間極值追蹤與趨勢過濾。
3. **Causal Execution**: 整合訊號並管理訂單生命週期（時段結束即出場）。
4. **Audit HUD**: 披露實時執行狀態與版本定標。

## 使用建議
- **推薦標的**: 指數、商品期貨、高波動率個股。
- **推薦時框**: 5M 或 15M 為執行時框。
- **版本號**: v26.0422.0601
