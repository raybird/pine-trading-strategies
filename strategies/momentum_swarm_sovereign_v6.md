# TN Momentum Swarm Sovereign v6 (動能蜂群主權版)

## 策略概述
此策略實體化了 TeleNexus 的「全局主權分配」規訓。它不侷限於單一標的，而是將多個關鍵資產（如 BTC, GOLD, USOIL, DXY）視為一個動態的「蜂群（Swarm）」。透過即時運算蜂群內部各成員的因果動能位移 (Rate of Change)，系統會自動將 100% 的主權資金輪動至動能最強的領袖資產中，實現對全球流動性溢價的自動化捕獲。

## 核心特性
- **多資產動能審計 (Swarm ROC Engine)**: 透過自定義 `method` 同步獲取五個不同領域標的的物理價格數據，並進行歸一化的動能百分比演算。
- **陣列自動排名 (Array-based Ranking)**: 利用 v6 `array.max` 與 `array.indexof` 方法，在每個輪動週期自動鎖定當前蜂群的「領袖標的」。
- **三級輪動規訓 (Rotation Speeds)**:
  1. **Aggressive (5 Bars)**: 極短線輪動，捕捉日內高頻熱點切換。
  2. **Balanced (10 Bars)**: 標準規訓模式，過濾短期隨機波動。
  3. **Sovereign (20 Bars)**: 主權級趨勢輪動，追求穩定的長週期溢價。
- **資產主權對位**: 當系統發現新領袖時，會自動清算舊資產並建立新主權頭寸，確保資金的高流動性與因果一致性。

## 模組說明
1. **Asset Sensor**: 負責多符號數據的動態對焦與獲取。
2. **Momentum Auditor**: 執行高精度的 ROC 因果演算。
3. **Rotation Logic**: 負責決策蜂群領袖並觸發 Git 級別的執行標籤。
4. **Swarm Dashboard**: 即時披露全蜂群的動能排名與當前主權歸屬。

## 使用建議
- **主週期**: 建議在 4H 或 Daily 週期使用以降低交易摩擦。
- **資產組合**: 進階使用者可自行定義 `s1` 到 `s5` 的符號，建議選擇具備負相關性的標的（如 黃金對位美金）。
