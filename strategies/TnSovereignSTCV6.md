# Tn Sovereign STC v6

## 策略概述
這是一款基於 **Schaff Trend Cycle (STC)** 謝夫趨勢週期深度重構的主權策略，採用 Pine Script v6.0 標準構建。其核心理念在於結合 MACD 的趨勢感應與雙重平滑隨機指標的週期捕捉，精確定位市場「週期性翻轉」的物理臨界點。系統透過監測 STC 指標在極端區域後的極性轉向，並要求其獲得 200 EMA 宏觀地板支持，確保交易指令具備高度的因果穩定性與趨勢一致性。

## 核心特性
- **雙重平滑週期規訓 (Cycle Audit)**:
    - 採用 STC (10, 23, 50) 組合，具備極佳的趨勢敏感度與週期識別力。
    - 透過雙重平滑演算法過濾掉傳統隨機指標的隨機噪音，達成對週期極點的精準捕捉。
- **極端區域審計 (Extreme Zone)**:
    - 定義 25 與 75 為執行邊界。只有在指標從極端超賣區回升（做多）或從極端超買區回落（做空）時，才會產出主權信號。
- **宏觀環境規訓 (Regime Audit)**:
    - 強制整合 200 EMA 作為趨勢地板，自動過濾掉逆向操作，確保所有週期交易均具備深厚的宏觀支持。
- **實時審計 HUD**: 在圖表右上角即時披露 STC 數值、週期極性、環境偏見以及實時波動率，達成執行主權的完全透明化。

## 模組說明
1. **STC Config**: 負責謝夫週期的參數與極端門檻規訓。
2. **Audit Layers**: 執行趨勢地板過濾與 ATR 風險對位。
3. **Causal Signals**: 整合訊號並管理訂單生命週期。
4. **Audit HUD**: 披露實時執行狀態與版本定標。

## 使用建議
- **推薦標的**: 指數、趨勢型藍籌股、外匯波段交易。
- **推薦時框**: 1H (執行) 搭配 4H (環境)。
- **版本號**: v26.0426.0632
