# TN Liquidity Sweep Sovereign v6 (流動性掃蕩與動態規訓版)

## 策略概述
此策略致力於識別「市場操縱」的物理邊界。它透過監控樞紐高低點 (Pivot High/Low) 的清掃行為，捕捉價格在虛假突破後的迅速回歸。不同於傳統策略，它結合了「主權生命線 (EMA 200)」的因果過濾與動態 ATR 追蹤止損，確保系統在高波動環境下仍具備極強的生存能力。

## 核心特性
- **主權生命線對位 (Life Line Alignment)**: 所有獵取信號必須與日線級別的趨勢方向一致，確保執行動作具備宏觀因果支持。
- **結構化掃描 (Sweep Logic)**: 精確偵測價格對 Pivot 邊界的「穿刺並回歸」行為，這是識別機構級止損獵取的核心邏輯。
- **動態追蹤止損 (Dynamic Floor)**: 採用 v6 變量管理實作的「最高價/最低價回撤止損」，基於 ATR 自動調整物理防禦邊界。
- **實時執行儀表板**: 透過 Table API 即時披露當前的市場環境 (Regime) 與生命線位置，提升執行的透明度。

## 使用建議
- **主週期**: 30M、1H。
- **流動性回顧**: `lookback` 建議設為 20 以上以捕捉具備結構價值的邊界。
- **版本號**: v26.0418.0715
